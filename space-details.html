Space Invaders

Project Overview

My partner and I implemented the classic space invaders arcade game implemented using an arduino mini and ADAfruit RGB matrix. The player moves the "spaceship" with a potentiometer and fires with a button.

Unfortunately the game can only be played with an Arduino Uno and an RGB LED Matrix as of now; however, I plan to build a JavaScript simulation for live demos.

Date: Winter 2019

Project Role

My partner and I worked together in the same IDE (i.e., pair programming) to implement the game and all functionality.

Relevant Technology

  • Arduino Uno
  • Arduino IDE
  • C++
  • RGB Matrix LED Board

Relevant Skills
