Gravity

Project Overview

Gravity is a 2D puzzle-platform game where the player can use the jetpack, laser gun, and black holes to overcome obstacles and reach the portals at the end of the levels to advance through all six. The game is not quite fully complete nor is it bug-free as prioritizing tasks to complete (triaging) in a strictly time-constrained environment was part of the project specification as we were only given two weeks to submit a final deliverable.

Date: Fall 2021

Project Role

As this was a solo development project, I was responsible for the implementation of all aspects of the game from sprites, to mechanics, to music and effects.
